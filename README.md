# テーマ授業【ログイン】実践ワーク

* Ruby version 2.3.0
* Rails version 4.2.5.1


### Rails4.2.5.1をインストール

```
gem install rails --version="4.2.5.1"
```

### クローンする

```
git clone https://github.com/YujiMiyashita/login_haihu.git
```

### 必要なgemをインストールする

```
bundle install
```

### PostgreSQLを起動させる

```
sudo service postgresql start
```

### DBを作成

```
rake db:create
```
